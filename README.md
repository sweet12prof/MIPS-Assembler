# MIPS-Assembler
Assembler for built Processors

Consists of back-end modules for the assembler middleware. 
The assembler translates up to 15 instructions they include:
  1. add
  2. addi
  3. sub
  4. and 
  5. or 
  6. xor
  7. slt 
  8. sll
  9. srl
  10. beq
  11. lw
  12. sw
  13. j
  14. jal
  15. jr
